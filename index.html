Your clipboard on the Net

EZCP.io is a *clipboard for remote files*. Install the CLI of your choice. Then visit http://ezcp.io to *get a single-use token*.

COPY

On machine A, copy your file to the clipboard

ezcp [file path] [Token]

Or use pipes

[any command] | ezcp [Token]

PASTE

On machine B, copy from the clipboard to a new file

ezcp [Token] [file path]

Or use redirects

ezcp [Token] > [file path]

Or even use curl

curl -L https://ezcp.io/download/[Token] > [file path]

Note: all traffic is *encrypted with TLS* but you can use AES encryption with the -x [passphrase] option on both sides.
We won't look at your files anyway: we take your privacy very seriously.

Easy to install

Our clients are Open Source . Feel free to *review our code*, send Issues and Pull Requests!

Easy install from binary
wget -o ezcp https://ezcp.io/linux
wget -o ezcp https://ezcp.io/osx
wget -o ezcp.exe https://ezcp.io/windows

Or simply *click the link* in a browser

or with node.js
npm install -g ezcp

Free tier

  • Tokens can be used only once to copy, and once to paste
  • When a file is copied to a token, it is available for one hour

Do you need more? Read below...

  • Once pasted, a token is no longer usable
  • 20MB File size limit

Premium Year

Use ezcp -b to obtain an address, and send 0.01 BTC or more to it. Note the bitcoin transaction Hash and use it in ezcp -l [hash] to log in.

You can now enjoy a *full year* of ezcp with a permanent token, one that doesn't change and can be reused
plus you can upload large files (*150MB* vs. 20MB)
plus your files remain in the clipboard *up to 24h* instead of 1h...

Plus you're helping ezcp.io develop and host an amazing service!
Thank you for that.

*No need for a token anymore*. To copy a file to the clipboard

ezcp [file]

and to paste it use

ezcp [file path]

Use your permanent token in batches, CD pipelines, etc...

After one year, your token expires but you can re-do the --bitcoin and --login process.
